# BlockTemplates
Better Rails templates.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'block_templates'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install block_templates

## Usage

In terminal type.

`$ rails g model company`
